Autocomplete Identifier Bookmarklet

WCAG 2.1 success criteria 1.3.5 states that the purpose of each text box should be noted in the code. This is commonly done via the autocomplete attibute. Unfortunately there isn't an easy way to determine if this attribute has been set without viewing the code. This bookmarket will display the value of the autocomplete attribute next to the text box.

Description

This is simple bookmark will display the value of the autocomplete values so you don't have to view the HMTL code.

Autocomplete identifier bookmarklet

Add a bookmarklet

To add the bookmarklet to your browser, drag the bookmarklet button above to your bookmark or favorite bar.

To activate the bookmarklet on another webpage, just click the bookmarklet in the favorite bar.

Note about Chrome (keyboard-only instillation)

There is a defect in some versions of Chrome that doesn't allow you drag a bookmarklet into the favorite bar. To install a bookmarklet in Chrome:

  1. Copy the javascript code.
  2. Under the "Bookmarks" menu, choose "Bookmark Manager".
  3. From the organize menu button to the right of the search box, choose "Add new bookmark".
  4. Add the bookmark title into the "name" field.
  5. Paste the javascript code into the "url" field.
