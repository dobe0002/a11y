Bootstrap 4 Modal

Out of the box, there appears to be a few issues with the accessibility of the Bootstrap 4 modals. This jQuery based script fixes these issues and makes calling the modal easier.

Description

When a user closes a modal, they should be brought back to the item (link, button, etc.) that initiated the opening of the modal window. When using javascript to open a modal window, the initiating item appears to be lost and the user is returned to the top of the page when the modal is closed.

This script does three things. First it adds the modal HTML if it doesn't already exist. Secondly it creates an easier way to set the modal content. Lastly it keeps track of the item that initiated the opening of the modal and returns the user to the item.

Requirements

  • Bootstrap 4 JavaScript
  • Bootstrap 4 CSS
  • jQuery
