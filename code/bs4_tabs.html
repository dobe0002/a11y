Bootstrap 4 Tabs

Out of the box, Bootstrap 4 bas are not compliant with WCAG 2.0. This jQuery based script fixes these issues.

Description

According to the ARIA authoring guide, a user should only use the arrow keys to move between tabs. Out of the box, BootStrap 4 tabs forces users to use the "tab" key to select the next tab.

This script disables using the tab key to get to the next tab and instead allows the user to use the arrow keys.

Using the arrow key may be confusing for some visual users, so a message notifying them how to use the tabs would be beneficial.

Requirements

  • Bootstrap 4 JavaScript
  • Bootstrap 4 CSS
  • jQuery
