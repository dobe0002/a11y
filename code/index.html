My day job is that of a web developer. Below are a list of my more generic accessible code snippets. There may not be a lot here yet, but I will slowly add them as time goes on.

Feel free to use the snippets in your own projects.

Code snippets

    Change the default underlining of links so it is more accessible to all users.

  • Bootstrap 4 Tooltips

    Modify Boostrap 4 tooltips to comply with WCAG 2.1.

  • Bootstrap 4 Tabs

    Modify Boostrap 4 tabs to use the arrow keys.

    This is a modified version of the main SkipTo plugin that removes some false positives in WAVE. Kim's Fork of SkipTo

Bookmarklets

    Bookmarklet that displays the autocomplete attribute making testing WCAG 2.1's input purpose success criteria easier.
